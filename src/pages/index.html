NovaDine provides digital ordering solutions to regional and national restaurant chains. Offerings include online ordering via mobile app and web browser, featuring patented group ordering technology and curbside check-in for an unmatched customer service experience. I redesigned and rebuild NovaDine's corporate website from the ground up in the fall of 2016, with an unmitigated focus on presenting site content in a simple, elegant package. All-Times Search

All-Time(s) Search is a project built on Google's AngularJS JavaScript framework, enabling a single page application experience. It utilizes CSS Flexbox and JavaScript's built-in math functions to generate a virtual newspaper feel, generating a layout that is truly unique with each and every keywork search. The New York Times articles API is used to search for and deliver all data, providing results that date all the way back to the daily's beginning on September 18, 1851.

  • NYT Web API
